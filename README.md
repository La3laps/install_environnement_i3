# Install i3 environnement

## Description

Installation of my i3 environnement on Ubuntu at LeCampusNumérique : taken from [TheDistroHopper/i3wm-nord](https://github.com/TheDistroHopper/i3wm-nord)

## Installation

### Prerequisites

Install i3 on Ubuntu by running :

- `sudo apt install i3`
- Clone the repo using :
  `git clone https://github.com/La3laps/install_environnement_i3.git`

### Steps

1. Log out of your session
2. Login again chosing the i3 window manager
3. `win + enter` or `alt + enter` to open a terminal
4. Find the folder "install_environnement_i3" where you cloned
5. Launch the install_env.sh script by running :
   `./install_env.sh`
6. Follow the steps and voilà !!
